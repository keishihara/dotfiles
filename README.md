# dotfiles

**Dependencies**
- coreutils (macOS)
- git, curl
- tmux, [tpm](https://github.com/tmux-plugins/tpm)

**Shell**
- macOS > zsh
- Ubuntu(rootless) > bash

## Installtion
```bash
$ cd && git clone https://github.com/keishihara/dotfiles
$ cd dotfiles
$ source install.sh
```
